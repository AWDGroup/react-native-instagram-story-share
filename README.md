# React Native Instagram Story Share #

## Installation ##


`yarn add react-native-instagram-share`

### iOS ###

`pod install ios`

### Android ###

`yarn link react-native-instagram-share`


## Usage ##

```
RNInstagramStoryShare.share({
	backgroundImageUrl: '',
	backgroundTopColor: '',
	backgroundBottomColor: '',
	deeplinkUrl: '',
	stickerImageUrl: '',
})
```
